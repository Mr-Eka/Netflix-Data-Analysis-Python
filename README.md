# Netflix Data Analysis Using Python

## Project Overview
This project performs Exploratory Data Analysis (EDA) on the Netflix dataset using Python.  
The goal is to analyze Netflix content and uncover insights related to content type, release trends, and country-wise distribution.

## Dataset
- Dataset: Netflix Movies and TV Shows
- Source: Kaggle
- Records: 8,800+ titles

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Analysis Performed
- Data cleaning and handling missing values
- Exploratory Data Analysis (EDA)
- Data visualization using multiple chart types
- Insight generation from visual patterns

## Visualizations Included
- Count Plot (Content Type)
- Bar Chart (Top Producing Countries)
- Histogram (Release Year Distribution)
- Box Plot (Release Year by Content Type)
- Trend Chart (Content Added Over Years)
- Heatmap (Correlation)

## Key Insights
- Netflix’s catalog is dominated by movies.
- The platform focuses heavily on recent content.
- The United States and India are the largest content contributors.
- Netflix content additions increased significantly after 2016.

## Conclusion
This analysis highlights Netflix’s content strategy and growth trends using Python-based EDA techniques.  
The insights derived can support data-driven decisions in content planning and market analysis.
